## 📖 Tentang Proyek

**Pembelajaran IPS** adalah media pembelajaran digital berbasis web yang menyajikan materi Ilmu Pengetahuan Sosial (IPS) untuk siswa Sekolah Dasar secara interaktif dan menarik. Website ini dibangun menggunakan HTML tanpa framework tambahan, sehingga ringan dan mudah diakses dari berbagai perangkat.

### Tujuan
- Menyediakan media belajar IPS yang mudah diakses oleh siswa SD
- Mendukung guru dalam menyampaikan materi secara digital
- Menyajikan konten pembelajaran yang interaktif dan menyenangkan

---

## ✨ Fitur

- 📌 Materi IPS lengkap sesuai kurikulum SD
- 🖼️ Tampilan visual yang menarik dan ramah anak

- ⚡ Ringan dan cepat diakses tanpa perlu koneksi internet cepat
- 🔍 Navigasi yang mudah dipahami oleh siswa SD

## 🚀 Cara Menggunakan

### Opsi 1 – Buka Langsung di Browser
1. Clone atau download repository ini
2. Buka file `index.html.html` menggunakan browser (Chrome, Firefox, Edge, dll.)
3. Website siap digunakan tanpa instalasi tambahan

### Opsi 2 – Clone via Git
```bash
# Clone repository
git clone https://github.com/hendra01HS/pembelajaranIPS.git

# Masuk ke folder
cd pembelajaranIPS

# Buka file di browser
start index.html.html      # Windows
open index.html.html       # macOS
xdg-open index.html.html   # Linux
```

### Opsi 3 – Akses via GitHub Pages
Kunjungi link demo di atas langsung dari browser tanpa perlu mengunduh apapun.

## 📋 Materi yang Tersedia

Website ini mencakup materi IPS untuk jenjang Sekolah Dasar, antara lain:

- 🗺️ **Lingkungan Sekitar** – Mengenal lingkungan rumah, sekolah, dan masyarakat
